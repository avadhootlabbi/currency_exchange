# 💱 Currency Converter

A modern and responsive **Currency Converter** web application built with **Python** and **Streamlit**. It provides real-time currency conversion using live exchange rates with a clean and user-friendly interface.
## ✨ Features

- 🌍 Real-time currency exchange rates
- 💰 Convert between multiple international currencies
- ⚡ Fast and responsive Streamlit interface
- 🎨 Modern dark-themed UI
- 📱 Responsive design
- 🔄 Accurate mid-market exchange rates

---

## 🛠️ Tech Stack

- **Python 3**
- **Streamlit**
- **Requests**
- **Exchange Rate API**
- **Pandas** (if used)
 
## 🌎 Supported Features
